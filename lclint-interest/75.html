I'm currently writing a review for a German computer magazine
that compares the SVR4 standard lint with Lclint and Gimpel's
FlexeLint.

about an ignored return value of a locally declared function with
"int"-result, but it doesn't complain in case of printf.  I suppose
this is intended behavior, as many programs ignore printf's return
