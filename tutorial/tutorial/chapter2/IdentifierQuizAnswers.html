Answers to Identifier Quiz

1stPlace Not valid. Starts with a number.
Float Valid. (remember C is case sensitive!)
minVertex Valid.
register Not valid.register is a keyword
best-match Not valid. Can't have a dash.
__maximum Not (really) valid. This will compile unlike the others, but variables begining with '_' are reserved for use in the C Standard Library so don't use identifiers like this.
First&Last Not valid. & is not allowed in identifiers.
FirstBe4Last Valid.
