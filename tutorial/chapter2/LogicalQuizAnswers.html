Answers to Logical Operators Quiz

  1. Given that: 
     int a = 0; 
     int b = 1; 
     int c = 1;
    Evaluate: 
     (a && b) || c
    Answer: 1
  2. Given that: 
     int a = 35; 
     int b = -4; 
     int c = 0; 
     int d = 5;
    Evaluate: 
     ((b < a) && c) || d
    Answer: 1
  3. Given that: 
     int a = -35; 
     int b = -4;  
     int c = 0;   
     int d = 5;
    Evaluate: (in precedence what comes first && or ||?) 
     (b < a) && c || d
    Answer: 1
  4. Given that: 
     int a = -35; 
     int b = -4;  
     int c = 0;   
     int d = 5;
    Evaluate: 
     (b < a) < c < d
    Answer: 1
  5. Given that: 
     int a = 14; 
     int b = -1; 
     int c = 1;  
     int d = 3;
    Evaluate: 
     ((-b == c) >= d + 13) + 13 >= a
    Answer: 0
